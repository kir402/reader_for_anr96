# reader_for_anr96

Данный проект был создан для внтренних нужд комнании ООО "ИТМ". Цель этого проекта непрерывный сбор данных с анализатора электрической сети ANR96 фирмы ABB и запись этих данных в файл для дальнейшего анализа. Сбор данных ведётся через интерфейс RS485 по протоколу MODBUS RTU. Доступ к интерфейсу осуществляется через COM порт ПК. Во время работы программы использовался преобразователь USB/RS485. 

При запуске программы происходит поиск всех доступных COM-портов, так же этот поиск выполняется при нажатии кнопки "Обновить список". 
Кнопка "Проверить порт" служит для предварительной проверки доступности выбранного порта. 
Программа позволяет задать адрес устройства для опроса. При нажатии кнопки "Создать файл" начинается опрос: происходит подготовка пакетом MODBUS для опроса, создаётся файл в который первой строкой заносится титул для значений и запускается таймер.
Опрос устройства выполняется при срабатывании таймера. Первым значением в строке записывается текущее время. Затем идёт запрос на устройство и запись полученных данных в файл. Для каждого пакета запроса эмпирически было выбрано время ожидания ответа (файл ANR.h).

В полночь файл закрывается и открывается новый файл с именем новой даты.

Данная программа проработала на двух ноутбуках в течении необходимого срока наблюдения (месяц).
